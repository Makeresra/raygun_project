### ğŸ“ About the Project
This project is a fully functional replica of the legendary **Ray Gun Mark 2** from **Call of Duty: Zombies**. The 3D-printed shell is brought to life using an **Arduino Nano** microcontroller.

It features realistic **RGB LED** animations that mimic the in-game effects and a rechargeable Li-Po battery system managed by a **TP4056** module.

### âœ¨ Features
* **Firing Effect:** Realistic green/blue burst animation when the trigger is pulled.
* **Idle Mode:** Breathing animation (slowly pulsing green light) when not firing.
* **Rechargeable:** Built-in Li-Po battery chargeable via USB Type-C or Micro-USB.
* **Safety:** Dedicated main power switch and battery protection circuit.

### ğŸ› ï¸ Hardware List
* **Microcontroller:** Arduino Nano
* **Power:** PX1300XP-40C Li-Po Battery (or any 3.7V LiPo)
* **Charging:** TP4056 Charging Module
* **Lighting:** 3x RGB LED (Common Cathode)
* **Controls:** 1x Tactile Button (Trigger), 1x Toggle Switch (Main Power)
